# 📊 Gym Admin Dashboard

## 📘 Project Overview
Admin panel to track gym members and handle operations efficiently.

## 🛠️ Tech Stack
- Node.js
- Express
- PHPMyAdmin

## ✨ Features
- Separate login for admins
- Data visualization for gym usage
- Admin-based order handling
